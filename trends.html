溺水趋势

中国已大幅降低溺水死亡事故，尤其是儿童溺水，但人口老龄化意味着剩余风险正逐渐转向老年人。与此同时，海滩与离岸流仍威胁着年轻人的生命。

根据全球疾病负担研究（GBD 2023）的数据，中国的溺水死亡人数自1990年以来持续下降，从约15万人降至2023年的约4.3万人。

死亡人数与发生人数对应左侧坐标轴；伤残调整生命年（DALYs，百万）对应右侧坐标轴。

从1990年到2023年，溺水死亡人数下降了约71%（从约15万降至约4.3万），而溺水发生人数（包括获救者）下降了约49%（从约45万降至约23万）。死亡人数下降得更快，说明溺水不仅变得更少见，也变得更容易获救-每例死亡对应的发生人数从1990年的约1:3提高到2023年的约1:5。伤残调整生命年（DALYs）的降幅最大（约81%），表示死亡正逐渐从幼儿向老年人转移，因为每名儿童死亡所损失的健康生命年远高于老年人。

下降的原因

  1. 救生员数量和海滩管理水平提高
  2. 公众安全意识增强
  3. 急救和医疗响应速度加快

仍需警惕

虽然死亡人数下降，但溺水仍是中国儿童和青少年意外死亡的主要原因之一，老龄化使溺水风险向低防范意识的老年人转移。预防工作不能松懈。

全球 · 亚洲 · 中国 溺水死亡人数对比

原始死亡人数（2010–2023）。亚洲为东亚、南亚、东南亚及高收入亚太地区之和（不含中亚）。中国的数字同时包含在亚洲与全球总数之内。

亚洲占全球溺水死亡的一半以上（2023年约56%）。中国约占全球的15%、亚洲的27%。三条曲线都在下降，但中国下降得最快，因此中国在全球中的占比正在缩小，在一个整体溺水严峻的地区里，中国是少见人数下降案例，这表示中国名众的安全意识与海滩管理水平正在显著提高。需要注意：中国的数字同时包含在亚洲与全球之内，且此处的"亚洲"是四个地区之和（不含中亚），并非单一的官方地区。

按年龄划分的溺水死亡（中国，2010–2023累计）

与全球相比，中国的儿童占比更低（34% 对 41%），而老年人占比更高（20% 对 11%）——这反映了中国儿童溺水的大幅下降与人口老龄化。

在全球范围内，0–14岁儿童占溺水死亡的41%；而在中国，这一比例为34%，老年人（70岁以上）则占20%（全球为11%）。中国的溺水死亡比全球更偏向老年人。原因有二，且都能在数据中看到：中国的儿童溺水大幅下降（5岁以下死亡从2010年的约1.3万降至2023年的约0.34万），同时人口老龄化使剩余风险向老年人转移。在普及安全教育与海滩游玩时，需关注老年人群体。需注意：此处"儿童"为0–14岁（受GBD年龄分组限制），且饼图为14年累计的分布，并非逐年趋势。

数据与方法

数据来源：全球疾病负担研究（Global Burden of Disease, GBD 2023），美国华盛顿大学健康指标与评估研究所（IHME）。通过 GBD Results 工具导出（vizhub.healthdata.org/gbd-results）。

统一查询条件：死因 = 溺水（Drowning）；指标 = 人数（Number）；性别 = 两性合计；各图的地区、年龄与年份见下。

  • 趋势图（死亡 / 发生 / DALYs）：地区 = 中国；年龄 = 所有年龄；年份 = 1990–2023。DALYs 数值除以 1,000,000，以"百万"为单位显示。
  • 地区对比图：地区 = 全球、中国，以及"亚洲"；年份 = 2010–2023（受全球数据起始年限制）。"亚洲" = 东亚 + 南亚 + 东南亚 + 高收入亚太地区，逐年相加得出（不含中亚），并非 GBD 的单一官方地区。
  • 年龄饼图：地区 = 全球 / 中国；年龄段 = <5、5–14、15–49、50–69、70+；将 2010–2023 各年相加。儿童 = <5 + 5–14（即0–14岁）；成年人 = 15–49 + 50–69；老年人 = 70+。

说明与局限：所有数值为 GBD 的点估计（均值），未在图中显示其不确定区间（上下限）。"亚洲"为本项目自行汇总，可能与其他来源的"亚洲"口径不同。中国的数字同时包含在亚洲与全球总数之内，三者并非互相独立的人群。受 GBD 年龄分组限制，"儿童"以0–14岁界定，而非0–19岁。
