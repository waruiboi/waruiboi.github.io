离岸流

离岸流是从海岸向外流动的强劲水流，是海滩溺水的主要原因。

660 2010–2019年冲浪区溺水死亡
低 中国南方海滩游客对离岸流的认知程度

如何逃脱离岸流

  1. 保持冷静，不要与水流对抗
  2. 沿海岸线平行游泳，脱离水流
  3. 脱离水流后，再向岸边游回
  4. 如果精疲力竭，漂浮并呼救

水母

水母蜇伤是中国海滩最常见的海洋生物伤害，占所有海洋生物伤害的60%以上。

60.81% 海洋生物伤害中水母蜇伤占比
1.5亿 全球每年水母蜇伤人数

被水母蜇伤后如何处理

  1. 立即离开水中
  2. 不要用手直接触摸蜇伤部位
  3. 用海水冲洗，不要用淡水
  4. 用卡片刮去残留触须
  5. 如出现呼吸困难或严重过敏，立即就医

安全指南

#120 中国紧急求救电话
#12395 海上救援电话

下水前

  1. 只在有救生员的海滩游泳
  2. 检查当天的海况和天气预报
  3. 儿童必须有成人全程陪同
  4. 告知他人你的游泳计划

下水时

  1. 不要单独游泳
  2. 留意旗帜信号—红旗代表危险
  3. 不要在疲劳或饮酒后游泳
  4. 注意脚下，避免踩到水母或海胆
